# Kodbase
A web-based code editor with user authentication.

## Tech Stack
- Frontend: React + Vite + TailwindCSS + Monaco Editor
- Backend: Node.js + Express + MongoDB

## Setup
1. Clone the repository
2. cd backend && npm install
3. cd ../frontend && npm install
4. Add .env file in the backend directory based on .env.example
5. Start both servers using npm run dev
